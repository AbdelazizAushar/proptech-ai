# AI Automation & WhatsApp Commerce System

## PropTech AI

AI-powered platform connecting a frontend app to a WhatsApp bot via n8n automation.

## Structure

- `frontend/` — React.js browsing app 
- `n8n/` — All automation workflow JSONs

## Stack

- Frontend: React.js + Vite
- Database: Supabase
- AI: Groq
- Embeddings: Cohere
- Automation: n8n
- Messaging: WhatsApp Business API (Evolution API)
