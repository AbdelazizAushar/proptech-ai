# AI Automation & WhatsApp Commerce System

## PropTech AI

AI-powered platform connecting a frontend app to a WhatsApp bot via n8n automation.

## Structure

- `frontend/` — Next.js browsing app
- `n8n/` — All automation workflow JSONs

## Stack

- Frontend: Next.js
- Database: Supabase
- AI: Groq
- Embeddings: Hugging Face
- Automation: n8n
- Messaging: WhatsApp Business API
